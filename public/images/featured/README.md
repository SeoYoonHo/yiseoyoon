# Featured Background Images

## 사용법
- `background.jpg` - 메인 페이지 배경 이미지
- 권장 해상도: 1920x1080px 이상
- 지원 형식: JPG, PNG, WebP
- 파일 크기: 2MB 이하 권장

## 관리자 페이지에서 변경
1. `/admin` 페이지 접속
2. "Featured Background" 섹션에서 "Upload New Background" 클릭
3. 새 이미지 업로드
4. "Save Changes" 클릭

## 현재 설정
- 파일: `background.jpg`
- 제목: "Featured Artwork"
- 설명: "대표 작품이 배경으로 표시됩니다"
